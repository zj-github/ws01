配置文件

daemonize	yes							设置redis以后台进程模式运行
pidfile		/var/run/redis_端口号.pid	pid文件位置
port		端口号						redis监听端口号
dir			/var/redis/端口号			设置持久化文件存放位置

数据类型



递增数字
redis 127.0.0.1:6379> incr num
(integer) 1
redis 127.0.0.1:6379> incr num
(integer) 2
redis 127.0.0.1:6379> incr num
(integer) 3

redis 127.0.0.1:6379> set num 40
OK
redis 127.0.0.1:6379> get num
"40"

指定一次增加的数值
incrby num 3

减少指定的整数
decr key
decrby key 3

增加指定浮点数
incrbyfloat key increment


字符串类型
set key value
get key
strlen key 不存在返回0

一次设置多个键值对
mget key [key ...]
mset key value [key value ...]

位操作


散列类型
hset car price 500
hget car name bmw

car:2	color:白色 ,name: 奥迪,price,90万

判断一个字段是否存在
hexists car model
若存在返回1，否则返回0，key不存在也返回0

当字段不存在的时候赋值
hsetnx car model 2.0T

使字段值增加指定的整数
hincrby person:1 score 1

删除字段
hdel car price
